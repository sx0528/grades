# 📊 GradesTracker

**一个现代化的学生成绩追踪与可视化分析应用**

利用 Python 的强大数据科学生态，GradesTracker 帮助教师和学生轻松管理成绩数据，并通过直观的图表洞察学习趋势。

---

## 🚀 技术栈与环境要求

- **Python 版本**：3.12 或更高版本
- **核心框架**：
    - **Streamlit**：用于构建交互式 Web 界面。
    - **Pandas**：用于高效的数据处理与存储。
- **可视化库**：
    - **Matplotlib & Seaborn**：提供经典的统计图表。
    - **Plotly**：提供交互式、可缩放的动态图表。
- **打包工具**：**PyInstaller** (用于生成独立可执行文件)

## 💻 快速开始

### 1. 环境准备

请确保你的环境中已安装 Python 3.12+，然后通过 pip 安装依赖：
pip install pandas streamlit matplotlib seaborn plotly
